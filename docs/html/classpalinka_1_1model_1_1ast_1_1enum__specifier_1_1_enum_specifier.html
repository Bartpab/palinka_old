Detailed Description

    Represents an enumerator specifier

    <enum-specifier> ::= enum <identifier> { <enumerator-list> }
                    | enum { <enumerator-list> }
                    | enum <identifier>

    Implicit,
    <enumerator-list> ::= <enumerator> | <enumerator-list>, <enumerator>
